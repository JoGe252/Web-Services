## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Python
- **Package Manager:** Pip
- **Container Runtime:** Docker

### Installation

Build Web-Services from the source and install dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/JoGe252/Web-Services
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd Web-Services
    ```

3. **Install the dependencies:**

**Using [docker](https://www.docker.com/):**

```sh
❯ docker build -t JoGe252/Web-Services .
```
**Using [pip](https://pypi.org/project/pip/):**

```sh
❯ pip install -r Assignment1-B00118121/requirements.txt
```

### Usage

Run the project with:

**Using [docker](https://www.docker.com/):**

```sh
docker run -it {image_name}
```
**Using [pip](https://pypi.org/project/pip/):**

```sh
python {entrypoint}
```

### Testing

Web-services uses the {__test_framework__} test framework. Run the test suite with:

**Using [docker](https://www.docker.com/):**

```sh
echo 'INSERT-TEST-COMMAND-HERE'
```
**Using [pip](https://pypi.org/project/pip/):**

```sh
pytest
```
